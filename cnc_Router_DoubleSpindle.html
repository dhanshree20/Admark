Double Spindle CNC Router

The Double Spindle CNC Router is designed for high-efficiency operations by using two spindles that can work at the same time. This setup allows for faster processing of identical designs or dual-tool operations without manual tool changes. It’s especially beneficial in woodworking, signage, and bulk production environments where speed and symmetry are essential.

Technical Specifications

  • Working Area: X = 1300mm, Y = 2500mm, Z = 220mm
  • Operating System: ADMARK DSP CONTROLLER / USB
  • Model: 1325
  • Travelling Position Accuracy: +/-0.05mm
  • Working Voltage : Single Phase 380 V AC, 50 HZ
  • Spindle Speed : 0 – 18000 rpm
  • Spindle Power : 3.5 KW, Single Phase Double Spindle Air Cooled Spindle / Water Cooled Spindle
  • Max Travel Speed : 15000 / min
  • Max Work Speed : 1000 / min
  • Aluminum T – Slot bed, Regular Tool Set, Clamps
  • Command : Command Code G, *.uoo, *.plt, * .mmg
  • Software : Type 3 / Vetric Cam / Carveco / Artcam
