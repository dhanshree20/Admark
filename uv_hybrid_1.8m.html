1.8m UV Hybrid

The UV Hybrid 1.8m printer is a versatile and compact solution designed for small to medium-scale printing operations. It supports both rigid and flexible media types, delivering high-resolution prints with sharp details, making it ideal for signage, promotional items, and interior graphics.

  • Media Thickness : Max. < 50mm
  • Ink Type : UV Curing Ink
  • Curing System : LED Curing System
  • Speed: 34 sq/ft/hr (On 4 pass) / 96 sq/ft/hr (On 4 pass)

  • File Formal : TIFF / PDF / JPES / EPS / AL
