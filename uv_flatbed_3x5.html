5' x 3' UV Flatbed

The UV Flatbed 5' x 3' printer is ideal for printing directly on rigid materials like acrylic, glass, wood, and foam board. It offers high-resolution, full-color printing with excellent durability and precision for indoor and outdoor use. This size is perfect for businesses needing medium-format prints with a compact machine footprint.

Technical Specifications

  • Usage/Application: Printing
  • Model: UV 1610 (05 HEAD SYSTEM)
  • Print Head: EPSON i3200 U1 / RICOH GEN 5 / RICOH GEN 6
  • Print Width: 1600mm * 1000mm (5 * 3)
  • Print Mode : (4C/6C/4C + W/6C + W/4C + W + V/6C + W + V/8C + W + V)
  • Print Resolution : 720 * 1800 DPI
  • Speed: 130 sq/ft/hr (On 4 pass)
  • Media Thickness : Max. 150mm; > 150mm Available
  • Ink Type : UV Curing Ink
  • Data Connector :USB 2.0 / 3.0
  • Power : 220 VAC / (-+10%) Single Phase, 50/60 HZ
  • Power Consumption : 2.8KW
  • Machine Size :3.2m (L) * 1.7m (W) * 1.4m (H) / 1100KG
  • Media Type : Glass, Wood, Ceramic, Metal Sheet, PVC Sheet, Plastic 3C Product, Etc
  • Special Ink : Optional White OR Varnish / Premier / Bright Gold Thermo Print
